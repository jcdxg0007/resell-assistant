# 转卖助手 · 项目结构与架构速览

> 用途：断上下文 / 新会话快速对齐。读完这一篇就能知道「东西都在哪、数据怎么流、最近改了什么」。
> 最后更新：2026-05-30

---

## 0. 一句话简介

多平台（闲鱼 / 拼多多 / 小红书）二手转卖的自动化选品 + 上架 + 订单 + 客服一体化系统。

- **前端**：React 19 + Ant Design 5 + Vite + react-router 7（`frontend/`）
- **后端**：Python 3.12 + FastAPI + Celery（`backend/`）
- **数据库**：PostgreSQL（SQLAlchemy async + Alembic 迁移）+ Redis（Celery broker / 缓存）
- **自动化**：Playwright（闲鱼/小红书爬虫与发布）
- **PDD 采集**：家里 Windows + 物理手机的 **APP worker**（`worker/pdd_app_worker/`），不走服务端爬虫
- **部署**：Docker + Sealos(k8s) + GitHub Actions CI/CD，API 容器启动**自动跑迁移**

## 4. 核心数据流：采集与选品「解耦」（2026-05 重构）

这是最近最重要的架构变化，务必理解：

```
采集层（只爬原始、只展示）              选品层（单独打分页）
  闲鱼挂牌 ─┐                            ┌─→ 十维度选品页 /selection/ten-dim
  PDD 商品 ─┴─→ 多平台比价页              │   按关键词取两池原始数据
            (原始数据并排展示，无打分)   └─→ 清洗 + 三层打分 + 排序 + 落缓存
```

- **多平台比价页**：闲鱼用 `/selection/xianyu/raw`（原始挂牌，无评分）、PDD 用 `/pdd-runs/items`。只展示「爬到什么就是什么」。
- **十维度选品页**：on-demand 打分，结果落 `selection_analysis` 表缓存，同词秒开；「重新分析」强制重算。
  - **A 闲鱼端（100分）**：价格竞争力20 / 单品热度(想要数)25 / 商品相关性15 / 标题质量10 / 客单价10 / 闲鱼竞争度10 / 图片质量5 / 挂牌新鲜度5
  - **B PDD端（100分）**：价格竞争力20 / 单品热度(销量)25 / 商品相关性15 / 标题质量10 / 客单价10 / 货源正规度10 / 图片质量10
  - **C 跨平台套利（100分）**：价差幅度30 / 套利利润率30 / 需求侧强度20 / 可行性风险20 → 自动判方向（贵端=卖出端）+ 预估利润 + 推荐度
  - 打分逻辑全在 `services/selection/ten_dim_scoring.py`，纯函数；API 在 `selection.py` 的 `/ten-dim/*` 端点取数+落缓存。

## 5. PDD APP Worker（`worker/pdd_app_worker/`）

PDD 网页(H5)搜索 2026 起对游客返空且高风险，**已彻底删除 H5 爬虫**。现在唯一通道是家里的物理手机：

- 部署在家里 Windows，连物理手机（uiautomator2 + OCR）跑 PDD APP 搜索
- 后端通过 `/pdd-worker/*` 派任务、收结果，落 `pdd_search_runs.items`(JSON)
- 关键开关（`tasks/selection.py` 顶部）：`_PDD_DISABLED=True`（H5 永久禁用）、`_PDD_USE_APP_WORKER`（APP 通道开关）
- 采集节奏 / 全自动跑批参数：`/pdd-worker-config`，前端在「多平台比价」页配置

## 7. 部署与迁移（重要约定）

- **CI/CD**：`.github/workflows/build-and-push.yml` —— 推 `main` 触发，条件构建 + Docker 缓存 + 重启 Sealos。
- **容器入口**：`backend/entrypoint.sh`
  - `APP_MODE=api`（默认）：**先 `alembic upgrade head` 自动迁移**，再起 uvicorn。
  - `APP_MODE=celery`：直接起 celery worker+beat，**不迁移**。
- **结论**：正常推代码、API Pod 重启即自动迁移，无需手动。`|| echo` 会吞掉迁移错误，部署后建议瞄一眼 API Pod 日志确认迁移成功。
- **迁移头**：当前 head = `f2a3b4c5d6e7`（products.published_at）；其前是 `e1f2a3b4c5d6`（selection_analysis 表）。链条单一线性。

## 8. 已知约定 / 坑

1. **闲鱼商品的 `category` 字段 = 搜索关键词**（落库时存的搜索词），同词比价/取数都靠它过滤。
2. **闲鱼「想要数」存在 `Product.sales_count`**（借位），不是真销量。
3. **taobao / 1688 爬虫已禁用**（`_ALIBABA_1688_DISABLED=True`），相关跨平台价差维度当年空转，已在十维度里去掉。
4. **卖家信誉无数据**：闲鱼列表页不返回卖家信用，要真信誉得做详情页爬虫（未做）。
5. **`published_at` 旧行为 NULL**：要闲鱼重采后才回填，「挂牌新鲜度」维度对存量行按无数据降级。
6. 前端十维度页**维度动态渲染**（遍历 `dimensions` 数组），后端调维度无需改前端。
